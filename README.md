# KOPI IoT üçµ

*Komposter Pintar Untuk Rumah dan Warung Kopi*

Kopi IoT adalah platform inovatif yang memanfaatkan teknologi Internet of Things untuk mengelola ampas kopi secara efisien. Sistem kami membantu penjual kopi memantau tingkat ampas mereka dan memberi notifikasi kepada pengemudi saat waktunya pengumpulan, memastikan lingkungan yang lebih bersih dan mendorong praktik berkelanjutan.

### Persyaratan üíª
- Laptop
- Koneksi Internet
- Code Editor
- PHP versi >= 8.2
- MySQL
- Rangkaian IoT (terpisah)

### Langkah-langkah Instalasi ‚è¨
1. Clone repository
```bash
git clone https://github.com/Falrafa4/kopi-iot.git
cd kopi-iot/php_app/
```

2. Impor database MySQL
```sql
CREATE DATABASE kopi_iot;
USE kopi_iot;

-- Kemudian impor database lewat PHPMyAdmin atau lainnya
```

3. Jalankan PHP web
```bash
php -S localhost:8000
```

4. Buka di browser
```bash
http://localhost:8000
```
